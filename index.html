Terminal Game using Python: Hangman

In 2024, I began learning Python after hearing about SmoothAg. I used ChatGPT to create the initial game loop, turning words into letters into underscores & expanded that to be a more personalized, playable game. Added a difficulty selection screen which chooses harder words & gives you less chances to be wrong. Finally added a 'help' screen or 'quit' option for the terminal game to end when you like.
